# Unwiished Recomp

> An unofficial native PC port of the Wii version of Sonic Unleashed, created through static recompilation. Free & open source, made by a dedicated fan.

---

## What is this?

Unwiished Recomp is a project that aims to bring the Wii version of Sonic Unleashed to PC natively, without emulation. The goal is to provide a smooth, high-performance experience with full keyboard and controller support.

This project is inspired by [Unleashed Recomp](https://github.com/hedge-dev/UnleashedRecomp).

## Goals

- Native PC port of Sonic Unleashed (Wii) via static recompilation
- Full keyboard and controller support (no Wiimote required)
- Stable 60fps with high resolution support
- Restore unused/cut content as optional mods
- Free and open source forever
